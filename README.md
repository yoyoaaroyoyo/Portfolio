# Portfolio Website

## Overview

This is a personal portfolio website created as a prototype to showcase my work, skills, and projects. The site is built using only HTML and CSS, with **no JavaScript**, focusing on simplicity, structure, and clean design.

⚠️ **Note:** This project is still under construction and will continue to evolve.

---

## Features

* Fully static website (HTML & CSS only)
* Clean and minimal design
* Responsive layout (basic support)
* Sections for projects, about, and contact
* Lightweight and fast-loading

---

## Purpose

The goal of this project is to:

* Build a strong foundation in core web technologies
* Demonstrate layout and styling skills without relying on JavaScript
* Serve as a starting point for a more advanced portfolio in the future
